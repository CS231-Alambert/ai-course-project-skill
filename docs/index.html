六合一工作台

每个功能对应一个 Python 脚本，现在全在浏览器里

01

✍️ Markdown 编辑器

左右分栏，实时预览。支持完整 GFM 语法：表格、代码块、LaTeX 公式、Mermaid 图表。

generate_docx.py
02

🤖 AI 论文生成

填入项目证据 + DeepSeek API Key，AI 自动撰写引言、方法、实验、总结各章节初稿。

NEW · Web only
03

🔍 证据提取

填写项目技术栈、API 接口、数据库 Schema，生成结构化证据报告。可粘贴到论文「实验环境」章。

build_evidence.py
04

🖼 图片管理

拖拽上传实验图表，网格预览，一键生成 image-map.json 映射文件。

build_image_map.py
05

📊 字数统计

自动按章统计字数 + 进度条可视化。不足提醒补写，超标提示精简。

count_words.py
06

✓ 参考文献检查

自动检测重复编号、年份缺失、语言平衡、DOI 引用。输出格式化检查报告 + 待办清单。

check_references.py
