# AI 课程论文章节模式

以下是为"人工智能导论"类课程大作业定制的章节结构和写作指南。

---

## 推荐结构（5 章，~6000 字）

```
第1章  引言（800 字）
第2章  相关工作（1000 字）
第3章  方法（2000 字）
第4章  实验（1500 字）
第5章  总结与展望（500 字）
```

---

## 第1章：引言

**目标**：让读者 3 分钟内理解你做了什么、为什么做、做出来什么效果。

**必须包含**：
- 问题定义（一句话说清要解决什么问题）
- 为什么这个问题值得做（1-2 段）
- 本文工作（方法一句话 + 关键结果一句话）
- 论文结构（"本文组织结构如下：第2章……第3章……"）

**避免**：
- "随着人工智能技术的飞速发展……"（空洞开头）
- 长篇背景铺垫（属于第2章）
- 列出本文没做的内容

**示例开头**：
> 图像分类是计算机视觉的基础任务。现有方法在 ImageNet 上取得了超过 90% 的 top-1 准确率，但在小样本场景下性能急剧下降。本文针对小样本图像分类问题，提出了一种基于原型网络与自注意力机制的混合模型，在 miniImageNet 5-way 1-shot 设置下达到 68.3% 的准确率，较基线模型提升 4.2 个百分点。

---

## 第2章：相关工作

**目标**：展示你了解这个领域，并定位你的工作位置。

**结构建议**（3-4 个小节）：
- 2.1 传统方法（简要）
- 2.2 深度学习方法（重点）
- 2.3 与你最相关的 2-3 篇工作（详细对比）

**写作技巧**：
- 每段结尾评价方法的局限性 → 自然引出你的方法
- 用表格对比（方法/年份/关键思路/局限）
- 引用要新（近 3 年为主）、要全（中英文都有）

**避免**：
- 罗列式堆砌："A 做了 X。B 做了 Y。C 做了 Z。"
- 引用 Wikipedia / 博客 / CSDN（用原始论文）

---

## 第3章：方法

**目标**：让读者能复现你的工作。

**必须包含**：
- 3.1 问题形式化定义（数学符号 + 公式）
- 3.2 模型架构图（一张总览图）
- 3.3 各模块详细描述
  - 输入表示
  - 核心算法/机制
  - 损失函数
  - 训练策略
- 3.4 实现细节（框架、超参数、硬件）

**图表要求**：
- 模型架构图（必须，推荐 draw.io）
- 算法流程图（可选，复杂方法需要）
- 关键公式（编号，LaTeX）

**检查清单**：
- [ ] 每个符号在它首次出现时都有定义
- [ ] 模型图与文字描述一致
- [ ] 损失函数公式正确
- [ ] 超参数有具体数值和选择理由

---

## 第4章：实验

**目标**：用数据证明你的方法有效。

**必须包含**：
- 4.1 实验设置
  - 数据集（名称、规模、划分方式、来源）
  - 评价指标（准确率/F1/BLEU/... + 公式或定义）
  - 基线方法（至少 3 个，注明引用）
  - 实验环境（Python版本、框架、GPU型号）
- 4.2 主实验结果（表格 + 分析）
- 4.3 消融实验（证明每个模块都有用）
- 4.4 案例分析/可视化（2-3 个具体例子）

**表格规范**：
- 最优结果加粗
- 表头：方法 | 指标1 | 指标2 | ...
- 表注说明评估协议

**避免**：
- "从表中可以看出……" 后不写具体数字和分析
- 只说"我们的方法最好"，不分析为什么
- 消融实验只做一组

---

## 第5章：总结与展望

**目标**：回顾全文，诚实评价，指出未来方向。

**必须包含**：
- 本文工作总结（3-5 句话，写你做了什么、得到什么结论）
- 主要贡献（列表，2-3 点）
- 不足与局限（诚实，2-3 点）
- 未来工作方向（具体可操作，不是"继续改进模型"）

**示例不足描述**（好 vs 差）：
- ❌ "模型还有改进空间"
- ✅ "当前模型在 ImageNet-LT 长尾分布上准确率仅 42%，因为重采样策略没有从根本上解决尾部类别表征不足的问题"

---

## 附录（可选）

- 附录A：更多实验结果（完整表格、额外可视化）
- 附录B：关键代码片段（核心模型定义、训练循环）
- 附录C：图表源文件索引

---

## 格式速查

| 元素 | 格式 |
|------|------|
| 正文 | 宋体 小四（12pt），1.25 倍行距，首行缩进 2 字符 |
| 一级标题 | 黑体 三号（16pt），居中 |
| 二级标题 | 黑体 四号（14pt） |
| 三级标题 | 黑体 小四（12pt） |
| 图表标题 | 宋体 五号（10.5pt），居中 |
| 参考文献 | 宋体 五号（10.5pt），悬挂缩进 |
| 代码 | Consolas 9pt |
| 页边距 | 上下 2.54cm，左右 3.17cm |
| 公式 | Times New Roman 12pt，居中 |
